# زمان‌بندی پیامک پیامیتو — Payamito Scheduler

افزونه وردپرس برای ارسال خودکار و زمان‌بندی‌شده پیامک از طریق سرویس [پیامیتو](https://payamito.com) پس از تغییر وضعیت سفارش‌های WooCommerce.

---

## ویژگی‌ها

- **زمان‌بندی پیامک** — پس از تغییر وضعیت سفارش، با تأخیر دلخواه (دقیقه / ساعت / روز) پیامک ارسال می‌شود
- **پترن (خط خدماتی)** — از قالب‌های تأییدشده پنل پیامک با متغیر داینامیک پشتیبانی می‌کند
- **شورت‌کدهای سفارش** — مقادیر سفارش به صورت خودکار جایگذاری می‌شوند
- **پنل ادمین** — مدیریت قوانین زمان‌بندی بدون نیاز به کدنویسی
- **تست آنلاین** — ارسال پیامک آزمایشی مستقیم از داشبورد وردپرس

---

## پیش‌نیازها

| نرم‌افزار | نسخه حداقل |
|-----------|-----------|
| WordPress | 6.0       |
| WooCommerce | 7.0     |
| PHP       | 8.0       |
| افزونه SOAP در PHP | فعال |

## نصب

1. پوشه `payamito-schedule` را در مسیر `wp-content/plugins/` قرار دهید
2. از داشبورد وردپرس وارد **افزونه‌ها** شوید و افزونه را فعال کنید
3. از منوی **زمان‌بندی پیامک** وارد تنظیمات شوید

## پیکربندی

### ۱. اطلاعات پنل پیامک

پس از فعال‌سازی، از منوی **زمان‌بندی پیامک ← تنظیمات پنل** نام کاربری و رمز عبور پنل پیامیتو خود را وارد کنید. این اطلاعات به صورت امن در دیتابیس وردپرس ذخیره می‌شوند.

### ۲. تعریف قوانین زمان‌بندی

در بخش **قوانین زمان‌بندی خودکار**، برای هر قانون تعیین کنید:

| فیلد | توضیح |
|------|-------|
| وضعیت سفارش | وضعیتی که تغییر آن پیامک را فعال می‌کند |
| تأخیر | مقدار و واحد زمانی (دقیقه / ساعت / روز) |
| کد پترن | شناسه قالب پیامک از پنل پیامیتو |
| متغیرها | مقادیر پترن به فرمت `key:value;key2:value2` |

### ۳. شورت‌کدهای قابل استفاده در متغیرها

| شورت‌کد | مقدار |
|---------|-------|
| `{billing_first_name}` | نام خریدار |
| `{billing_last_name}` | نام خانوادگی خریدار |
| `{order_id}` | شماره سفارش |
| `{order_total}` | مبلغ کل سفارش |
| `{billing_phone}` | شماره تماس خریدار |

**مثال متغیرها:**
```
name:{billing_first_name};amount:{order_total};id:{order_id}
```

## تست ارسال

از بخش **تست ارسال پیامک** می‌توانید با وارد کردن شماره موبایل، کد پترن و متغیرها به فرمت JSON، عملکرد سرویس را پیش از تنظیم قوانین بررسی کنید.

```json
{"0": "علی", "1": "1025000"}
```
